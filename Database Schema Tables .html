🗄️ Database Schema

👤 Users

Column NameData TypeKeyDescription
USERIDINTPKPrimary Key
FULLNAMENVARCHAR(100)Customer Name
EMAILNVARCHAR(100)UNIQUENo duplicates allowed
PHONENAMENVARCHAR(11)Phone Number
CREATEATDATETIMEDefault: Current Date

📍 Addresses

Column NameData TypeKeyDescription
ADDRESSESIDINTPKPrimary Key

📂 ProductCategories

Column NameData TypeKeyDescription

📦 Products

Column NameData TypeKeyDescription
PRODUCTSIDINTPKIdentity(1,1)

📊 Inventory
